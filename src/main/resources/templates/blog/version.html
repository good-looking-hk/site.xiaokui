    年初

    具体也知道是什么时候，只知道那时看了很多别人的博客，平台有CSDN、开源中国，个人博客有纯洁的微笑、廖雪峰的博客、阮一峰的网络日志等，经过 自己的一番寻思，发现写博客要么在第三方博客平台上写，如CSDN；要么使用第三方平台来托管发布博客，如Jekyll，hexo。
    但是这两种方式我都不喜欢，是的，都不喜欢，都觉得麻烦，既然是我自己的博客，我的控制权限应该是最大的，我想让它是什么样就是什么样，于是 有了这个博客，准确的说，这是一个博客平台（可以推广到所有人）。

    6月份左右

    那是在大三上学期期末，博客平台有了一个雏形，简单的界面、简单的功能，上传和管理都是手工在本地和服务器之间进行文件和数据库操作的。
    依稀还记得那是采用spring boot + jpa + thymeleaf + bootstrap + 马克飞象，奥，还有win10。
    写作是这样一个流程

    • 先在win10上用马克飞象写好（支持在markdown里面直接以二进制的方式插入图片，那时爱不释手），导出html。
    • 手动删除html文件里面的无用标签，然后把这个文件复制到服务器应用的templates目录下。
    • 然后在数据库添加一条记录，手工上传就完成了。
    • 最后就是在一个做好的模板母页上引入特定上传文件，然后我就可以看到一个精美的html博客页面了，开心

    那是在大三下学期开学3到4周的样子，跟老师有了一个约定，做个完整的博客平台给他看。大概是这么个流程

    • 首先是前端后台管理框架的选择，一番选择后确定了admilte，中途又加入了layui和vue，用起来还不错。
    • 前端确定好了之后，后面所做的就是把我那些手工操作变成在页面上操作，这一部分需要不少的代码。
    • 考虑到可以做成一个平台，于是把用户管理、菜单管理、权限管理都加了上去，这一部分只是简单的实现。
